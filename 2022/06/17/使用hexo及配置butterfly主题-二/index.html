菜单栏设置

菜单栏似乎最多只支持二级菜单

链接菜单可以由./source/_data/link.yml补充信息，虽然./source/link/index.md也能添加

按下面设置即可，备注都有可选项

社交图标设置

本地搜索

安装搜索插件

执行命令，安装本地搜索插件

$ npm install hexo-generator-search –save

配置

在根目录下的_config.yml中配置

如果某些页面不想被搜索到可以在编写时front matter中添加indexing: false即可使该文章不被检索

数学(推荐使用KATEX)

准备工作

卸载对应的渲染器

配置根目录的_config.yml

编写公式

相关的书写格式可以参考支持格式

$x^2+y^2+Dx+Ey+F=0$

$\begin{align} a&=b+c \ d+e&=f \end{align}$

$\begin{Bmatrix*}[r] 0 & -1 \ -1 & 0 \end{Bmatrix*}$

图片设置

设置看主题配置文件_config.butterfly.yml即可，这里不做赘述

字数统计

安装插件

要实现字数统计需要安装hexo-wordcount插件，安装命令为npm install hexo-wordcount --save

未安装插件在生成博客时会报错

设置主题配置文件

图片描述

photofigcaption是用来设置是否 显示图片描述文字的，false则不显示文字，true则显示文字

推荐将该属性设置为true，如不需显示文字可在[]中留空即可

绘制图表

使用mermaid标签可以绘制Flowchart（流程图）、Sequence diagram（时序图 ）、Class Diagram（类别图）、State Diagram（状态图）、Gantt（甘特图）和Pie Chart（圆形图），具体可以查看mermaid文档
使用前修改配置文件_config.butterfly.yml

  pie
    title Key elements in Product X
    "Calcium" : 42.96
    "Potassium" : 50.05
    "Magnesium" : 10.01
    "Iron" :  5
  
   

journey
    title My working day
    section Go to work
      Make tea: 5: Me
      Go upstairs: 3: Me
      Do work: 1: Me, Cat
    section Go home
      Go downstairs: 5: Me
      Sit down: 5: Me
